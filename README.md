# 消失的數學家 · 乘法公式密室逃脫

> 一款以國中數學「乘法公式」為主題的**點擊探索式密室逃脫遊戲**（point-and-click escape room），
> 採視覺小說（Visual Novel）敘事，全場景與角色皆為 **AI 生成插畫**。
> 對應康軒國二上 1-1「乘法公式」：分配律、和的平方、差的平方、平方差。

---

## 一、專題簡介

| 項目 | 內容 |
|------|------|
| 主題 | 國中數學・乘法公式（108 課綱 國二上） |
| 目標對象 | 國中八年級學生（13–14 歲） |
| 類型 | 點擊探索式密室逃脫 × 視覺小說 |
| 遊玩時長 | 約 15–25 分鐘 |
| 技術 | 純前端 HTML / CSS / JavaScript（無框架、無後端、無 localStorage） |
| 美術 | 全部場景與角色為 AI（gpt-image-2）生成，動漫／輕小說風 |
| 音效 | Web Audio API 即時合成，無任何外部音檔 |

## 二、故事情境

深夜，一封信躺在你桌上——寄件人是失蹤半年的恩師 **魏教授**。
信裡只有一句：「解開我留下的公式，你就會找到我。」

循著地址，你走進教授的研究塔，大門在身後喀地鎖上。塔中四層，各封著一道乘法公式機關，
由教授以公式投影出的 **守關幻影** 看守。唯有真正讀懂公式的人，才能逐層登上塔頂、尋回教授的最後手稿與真相。

- **主角**：你（見習偵探，教授的學生）
- **魏教授**：以信件與回音登場，給予故事引導
- **守關幻影**：各層守關者，負責提示與診斷式回饋

## 三、關卡設計（難度遞增）

| 層 | 場景 | 公式 | 康軒頁 | 互動機關 |
|----|------|------|--------|----------|
| 1 | 書房 | 分配律 a(b+c)=ab+ac | P.7 | 拖曳拼合藍圖 → 抽屜三碼密碼鎖 |
| 2 | 實驗室 | 和的平方 (a+b)²=a²+2ab+b² | P.10 | 四塊銅板面積拼合（a²、ab、ab、b²） |
| 3 | 暗房 | 差的平方 (a−b)²=a²−2ab+b² | P.13 | 移動燭光照出算式 → 選正負號 → 常數項數字鎖 |
| 4 | 塔頂之門 | 平方差 (a+b)(a−b)=a²−b² | P.16 | 60 秒限時速算挑戰（101×99、103×97、52²、48²） |

通關後顯示 **教授的最後手稿**（四公式面積圖解總整理，兼學習總結），並依提示使用次數頒予稱號：
代數大師（0 次）／公式解密者（≤3 次）／見習偵探（更多）。

## 四、如何執行

本專案為**純前端靜態網站**，需透過本機伺服器開啟（因瀏覽器對 `file://` 直接開啟有跨檔限制）。

```bash
# 進入專案資料夾
cd 消失的數學家

# 方法 A：Python（多數電腦內建）
python3 -m http.server 8000
# 然後瀏覽器開 http://localhost:8000

# 方法 B：Node.js
npx serve .

# 方法 C：VS Code 使用 Live Server 擴充套件，對 index.html 按「Go Live」
```

> 建議使用 Chrome / Edge / Safari 最新版；桌機與手機皆可遊玩。建議配戴耳機體驗音效。

## 六、技術重點

- **資料與邏輯分離**：所有內容集中在 `data.js`，程式碼不含硬編內容，方便老師改題或擴充單元。
- **教學性錯誤回饋**：不只寫「錯誤」，而是針對常見迷思給診斷（例：把 (a+b)² 當 a²+b²、把差的平方當平方差）。
- **無 localStorage**：進度僅存記憶體變數，符合部署環境限制。
- **無外部媒體**：音效以 Web Audio 合成；質感（塵埃／暈影／顆粒）以 CSS/SVG 生成。
- **響應式 + 無障礙**：支援觸控、最小點擊區 44px、尊重 `prefers-reduced-motion`。

## 七、擴充方式（做成系列關卡）

在 `data.js` 的 `ROOMS` 陣列新增一關物件，並在 `puzzles.js` 依 `openHot` 分派新謎題函式即可，
例如日後擴充「因式分解」「平方根」單元，不需改動引擎。詳見 `docs/專案結構說明.md`。

## 八、素材與版權

- 場景／角色插畫：以 gpt-image-2 生成，供教學專題使用。

- 公式內容與常見迷思：參考康軒國中數學國二上教材 1-1 乘法公式章節。

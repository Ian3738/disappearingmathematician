# 教學設計文件

## 一、教學主題與課綱對應

- **單元**：國中數學・乘法公式（108 課綱，國二上，康軒 1-1）
- **對應能力指標**：
  - A-8-1 認識多項式與乘法公式。
  - 能以幾何面積模型理解 (a+b)²、(a−b)²、(a+b)(a−b) 的展開。
  - 能運用乘法公式進行簡便速算。

## 二、教學目標

| 層次 | 目標 |
|------|------|
| 記憶 | 記住分配律、和／差的平方、平方差四個公式的形式。 |
| 理解 | 以面積模型理解每個公式「為什麼成立」，而非死背。 |
| 應用 | 能用平方差／完全平方公式做整數速算（如 101×99、52²）。 |
| 診斷 | 辨識並修正兩大常見迷思（見下表）。 |

## 三、四關與教學設計對照

| 層 | 公式 | 教學設計意圖 | 遊戲機制如何達成 |
|----|------|--------------|------------------|
| 1 書房 | a(b+c)=ab+ac | 建立「面積分割＝分配律」的直覺 | 拖曳把長方形拼回 a·b 與 a·c 兩塊，親手看見面積相加 |
| 2 實驗室 | (a+b)²=a²+2ab+b² | 破除「首尾平方就好」的錯覺 | 四塊銅板鋪滿正方形，親眼看見「中間兩塊 ab」 |
| 3 暗房 | (a−b)²=a²−2ab+b² | 區辨「差的平方」與「平方差」 | 選正負號 + 代入 (x−3)² 求常數項，強化中間項變號 |
| 4 塔頂 | (a+b)(a−b)=a²−b² | 從理解走向速算應用 | 限時速算，逼學生把大數拆成 (整十±小數) |

## 四、常見迷思診斷對照（重點）

遊戲的錯誤回饋不只是「答錯」，而是針對迷思給出診斷式引導：

| 迷思 | 學生的錯誤 | 遊戲回饋（守關幻影） |
|------|-----------|----------------------|
| 完全平方漏中項 | (a+b)² 寫成 a²+b² | 「教授的筆記劃掉了這個答案：中間還有 **2ab**，想想面積模型裡那兩塊長方形。」 |
| 混淆兩個公式 | (a−b)² 寫成 a²−b² | 「這是『平方差 a²−b²』的長相，但眼前是『差的平方 (a−b)²』——兩者不同，再看一眼燭光下的圖。」 |
| 末項符號 | (a−b)² 末項寫成 −b² | 「差的平方：中間 −2ab，末項 +b²（因為 (−b)²=+b²）。」 |
| 常數項 | (x−3)² 常數項寫成 3 或 −3 | 「(x−3)²=x²−6x+9，常數項是末項 (−3)²=9。」 |

## 五、學習評量與差異化

- **三段式提示**：方向 → 具體 → 近乎答案；求助不阻擋前進，但記錄次數並影響結局稱號，鼓勵先自行思考。
- **結局稱號**：代數大師（0 次求助）／公式解密者（≤3 次）／見習偵探（更多），作為形成性評量的即時回饋。
- **線索筆記本**：已解關卡的公式自動歸檔，供隨時複習。

## 六、教學使用建議

1. **課堂導入**：講解面積模型前，先讓學生玩第 1–2 關，建立直覺再回到課本推導。
2. **迷思釐清**：以第 2、3 關的錯誤回饋作為全班討論素材。
3. **速算練習**：第 4 關可作為單元結束的即時測驗，計時營造挑戰感。
4. **回家延伸**：請學生仿照 data.js 的題目結構，自出一題速算並說明用了哪個公式。
